Frontend Performance focuses on creating fast, responsive web applications that deliver exceptional user experiences across all devices and network conditions. This category covers performance measurement, optimization techniques, and modern tools that help you build lightning-fast interfaces.

## Core Areas

**Performance Measurement**: Lighthouse, Web Vitals, RUM, synthetic testing, and performance budgets.

**Loading Optimization**: Code splitting, lazy loading, resource prefetching, modern bundling, and critical rendering path optimization.

**Runtime Performance**: Efficient state management, minimizing reflows, offloading work to Web Workers, and using modern browser APIs.

**Caching & Networks**: HTTP caching, Service Workers, request deduplication, and smart CDN usage.

Deliver delightful experiences by making your UI fast, predictable, and resilient under real-world conditions.
